# Backend - Gerenciador de Projetos IA

Esta pasta contém a API REST desenvolvida em **NestJS**, utilizando o **Prisma ORM** e conectada ao banco de dados **PostgreSQL**.

*   Para instruções passo a passo de como configurar as variáveis de ambiente, executar as migrações do banco e rodar o servidor de desenvolvimento, por favor acesse o **[README principal na raiz do projeto](../README.md)**.
*   A documentação interativa das rotas via **Swagger** estará disponível localmente em `http://localhost:3333/api` assim que o servidor for iniciado.
